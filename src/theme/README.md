# theme/

Design tokens and global styles for the application.

- **colors.ts** — Full color palette (backgrounds, text, accent, success, danger, chart colors)
- **spacing.ts** — Consistent spacing scale (xxs through xxxl)
- **shadows.ts** — Box shadows and glow effects
- **typography.ts** — Font families, sizes, weights, line heights
- **breakpoints.ts** — Responsive breakpoints and media query helpers
- **animations.ts** — Keyframe animations and transition presets
- **globalStyles.ts** — CSS reset, scrollbar styling, focus states
- **index.ts** — Barrel export for all theme tokens
