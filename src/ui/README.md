# ui/

Reusable UI primitive components. These are low-level building blocks used across all pages.

- **Button** — Primary, secondary, ghost, danger, success variants with size options
- **Card** — Container with optional interactive/highlighted states
- **Input / TextArea** — Text fields with error states and labels
- **Select** — Styled dropdown select
- **Checkbox** — Custom animated checkbox with label
- **Toggle** — Segmented control for binary choices (e.g. Solo/Team)
- **Badge** — Small status labels with color variants
- **Pill** — Toggle-able tag buttons for multi-select (categories)
- **Chip** — Removable compact tags (selected technologies)
- **Toast** — Animated notification system (success, error, warning)
- **Skeleton** — Shimmer loading placeholders (base, text, card)
- **Modal** — Overlay dialog with header, body, footer
- **SearchInput** — Input with search icon prefix
- **EmptyState** — Placeholder for empty data views
