# pages/LogEntry/

The main daily task logging page (home page).

- **LogEntryPage.tsx** — Page shell with date header, project selector, task list, and submit button
- **components/ProjectSelector.tsx** — Dropdown with inline "add new project" capability
- **components/CategoryPicker.tsx** — Pill-based multi-select for categories with inline add
- **components/TechSelector.tsx** — Grouped checkbox technology picker with search, sub-tech boxes, and inline add
- **components/TaskCard.tsx** — Single task form card containing all task fields
