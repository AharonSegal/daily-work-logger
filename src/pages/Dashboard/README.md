# pages/Dashboard/

Analytics and history page with charts, filters, and data export.

- **DashboardPage.tsx** — Page shell with export buttons, filters, charts grid, and entry history
- **components/StatCards.tsx** — Overview stat cards (days logged, total tasks, top tech, top project, this week)
- **components/Charts.tsx** — Six Recharts visualizations (tasks over time, tech usage, categories, solo/team, projects, sub-tech breakdown)
- **components/EntryHistory.tsx** — Reverse-chronological entry list grouped by day with expand/collapse
